# Assets Viewer (typescratcherAssets)

A web application for browsing and accessing images and sounds from the Scratch editor asset library.
Scratchエディタで使用されている画像・音素材をブラウズできるWebアプリです。

現時点で未完成。少しづつ仕上げていく予定です。

## 📦 Features (主な機能)
- Browse Scratch images (costumes, backdrops): --Scratchの画像（コスチューム・背景）の閲覧
- Play Scratch sounds: --Scratchの音声の再生 
- Get direct asset URLs: --素材URLの取得 
- Simple and fast interface: --シンプルなUI
